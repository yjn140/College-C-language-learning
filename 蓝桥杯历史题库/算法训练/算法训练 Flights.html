【问题描述】

LiLand 是一个国家，包含 n 个城市。城市的编号从 1 到 n。这个国家因为它的非常奇怪的交通运输系统而闻名。这里有许多单向的航班让你可以在两个城市之间旅行，但是这些航班的安排使得你一旦离开一个城市就永远不会再回来了。

在以前，每趟航班恰好飞行一个小时，但是最近由于 Lily 成为了交通运输系统的新经理，她想要改变一些航班的飞行时间。具体来说，她想要把某些航班的飞行时间改为 2 小时，使得所有从 1 号城市到 n 号城市的旅行路线都花费相同的时间。

【输入格式】

输入数据的第一行包含两个整数 n 和 m，表示城市的数量和航班的数量。

接下来 m 行，每行包含两个整数 ai 和 bi (1 ≤ ai < bi ≤ n)，表示城市 ai 到城市 bi 有一趟单向航班。数据保证存在至少一条从城市1到城市 n 的路径，且航班不会形成环，任意两个城市之间不会有多趟航班。

【输出格式】

如果不能满足 Lily 的要求，只输出一行"No"(不包含引号)。

否则，第一行输出"Yes"(不包含引号)，接下来的 m 行，每行输出一个整数 ansi (1 ≤ ansi ≤ 2)，表示每条航班新的飞行时间。你的输出顺序需要与输入的航班顺序相符。

如果有多解，输出任意一种即可。

【样例输入】

3 3

1 2

2 3

1 3

【样例输出】

Yes

1

1

2

【样例输入】

4 4

1 2

2 3

3 4

1 4

【样例输出】

No

【样例输入】

5 6

1 2

2 3

3 5

1 4

4 5

1 3

【样例输出】

Yes

1

1

1

【数据规模和约定】

对于20%的数据，2 ≤ n ≤ 10，1 ≤ m ≤ 20；

对于60%的数据，2 ≤ n ≤ 100，1 ≤ m ≤ 300；

对于100%的数据，2 ≤ n ≤ 1000，1 ≤ m ≤ 5000。
