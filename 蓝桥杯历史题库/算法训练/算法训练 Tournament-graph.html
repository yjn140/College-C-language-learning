【问题描述】

你要构造一个有N个结点的竞赛图，使得对任意两个结点u和v（u≠v），从u到v的最短距离不超过2。

竞赛图就是基图为无向完全图的有向图（每对结点之间有一条有向边相连，且无自环）。

【输入格式】

输入只有一个正整数N（3≤N≤1000），表示要求的竞赛图的结点数。

【输出格式】

如果满足要求的竞赛图不存在，则输出-1。

否则输出一个N行N列的01矩阵（0表示没有边，1表示有边），要符合竞赛图的条件。

【样例输入一】

3

【样例输出一】

0 1 0

0 0 1

1 0 0

【样例输入二】

4

【样例输出二】

-1

【数据规模与约定】

对于40%的数据，N≤10；

对于70%的数据，N≤300；

对于100%的数据，N≤1000。
